Licensing For Your .NET Application

NetLicensing makes it easy for you to create a licensing solution for your .NET Windows application. Clone our git repository on GitHub by running the following command below (or use the GIT GUI on Windows):

$ git clone https://github.com/iandun/NetLicensing.git

Product Keys

NetLicensing can easily verify product keys

Software Licensing

NetLicensing can easily create and verify software licenses that can't be used on more than one computer
